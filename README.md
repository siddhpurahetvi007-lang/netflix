# Netflix Clone 

A Netflix-inspired content details page built using **HTML5** and **CSS3**. This project recreates the look and feel of a Netflix show page with a video banner, episode listings, show details, recommendations, and subscription plans.

---

##  Features

- Netflix-inspired user interface
- Hero section with video banner
- Fixed navigation menu
- Glassmorphism design effects
- Horizontally scrollable episode cards
- Show details and cast information
- "You Might Also Like" recommendation section
- Subscription plans section
- Dark theme with blurred background effects
- Clean and modern UI

---

##  Built With

- HTML5
- CSS3
- Flexbox
- CSS Positioning
- Glassmorphism Effects
